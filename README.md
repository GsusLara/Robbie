# Requerimientos 
- Asegurate de usar una version de node superior a la 14

# iniciar el proyecto en IDE


1. instala los paquetes
$ npm install

2. crea la variable de entorno 
$ cp .env.example .env

3. inicia el servidor web para la vista previa de tus cambios
$ npm run start
